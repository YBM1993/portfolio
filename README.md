# Ыбрашев Бақытжан — STEM портфолиосы

Полароидтық альбом форматындағы ұстаздың портфолио сайты.

## Беттер
- `index.html` — басты галерея (полароидтар + lightbox)
- `characters.html` — сабақтар
- `about.html` — мен туралы

## Жергілікті іске қосу
HTML-файлдарды бірден браузерден ашуға болады. Бірақ суреттер мен дыбыс
дұрыс жүктелуі үшін **жергілікті сервер** арқылы ашқан жөн:

```bash
# Python 3
python3 -m http.server 8000

# немесе Node.js
npx serve .
```

Содан кейін `http://localhost:8000` ашыңыз.

## GitHub Pages-ке жариялау
1. Жаңа репозиторий жасаңыз (мысалы `portfolio`)
2. Барлық файлды жүктеңіз (`index.html` түбірде болуы керек)
3. **Settings → Pages → Source: `main` branch / root**
4. `https://<username>.github.io/portfolio/` сілтемесінен ашылады

## Сурет қою
- Полароидтың үстіне суретті сүйреп тастаңыз (drag & drop)
- Немесе слотты басып, файл таңдаңыз
- Сурет `.image-slots.state.json` файлында сақталады — оны міндетті
  түрде репозиторийге қосыңыз

## Файлдар құрылымы
```
├── index.html                  # Басты галерея
├── about.html                  # Мен туралы
├── characters.html             # Сабақтар
├── main.js                     # Галерея логикасы
├── image-slot.js               # Сурет слоттары
├── .image-slots.state.json     # Жүктелген суреттер
└── resources/
    ├── gothic_ambient.mp3      # Фондық музыка
    └── paper_flip.mp3          # Парақ бұру дыбысы
```
